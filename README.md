## About

*A personal RESTFul/SPA blog engine made using a full Javascript stack.*

Tools and frameworks that may be used on this project:

- NodeJS
- MongoDB
- Redis
- Memcached
- Express
- EmberJS
- Bootstrap
- ZeroMQ

Mongo + Redis + ZeroMQ + Memcached + Node + Express + Ember + Foundation ou Bootstrap
